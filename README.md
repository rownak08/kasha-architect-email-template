# Responsive HTML Email Template – Architect Group

A **responsive, email-client-safe HTML email template** designed for an **Architecture / Design Studio** brand.  
Built using **table-based layouts** and **inline-friendly CSS** to ensure compatibility across major email clients.

“Live Preview: https://rownak08.github.io/kasha-architect-email-template/"

---

## 📌 Features

- Table-based HTML layout (email safe)
- Responsive design (desktop & mobile)
- Cross-client compatible (Gmail, Outlook, Yahoo, Apple Mail)
- Google Fonts (Poppins)
- Three-column content section
- Promotional CTA with coupon code
- Social media footer
- Clean architectural branding style

## 🚀 How to Use

1. Open `index.html` in a browser to preview.
2. Replace images inside the `img/` folder with your own.
3. Update text content for your architecture firm.
4. Inline CSS if required by your email service provider.
5. Upload HTML directly to:
   - Mailchimp
   - SendGrid
   - Brevo
   - Campaign Monitor
   - Custom SMTP systems
